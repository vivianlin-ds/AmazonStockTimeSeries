# AmazonStockTimeSeries
Practice DS project with Amazon Stock Dataset from Kaggle.

Goal: Predict the daily return for Amazon Stock in 2026.

## Data Source
[Kaggle Amazon Stock Prices 1997-2025](https://www.kaggle.com/datasets/ibrahimshahrukh/amazon-stock-prices-1997-2025-amzn-ohlcv?select=AAPL_Stock_Price_Dataset.csv)
